TUI Index

The Text User Interface package contains widgets for use on the command line. Objects in the package use the tui. namespace.
  • BooleanInput.md - Sublasses `tui.Input` and only permits `y` or `n` to be entered by the user.
  • ComboInput.md - Provides similar functionality to a combo box widget.
  • DateInput.md - Sublasses tui.Input and only permits date to be entered in various forms.
  • EditorInput.md - A class that follows the tui style but launches vi or the users preferred text editor to fetch input.
  • FileInput.md - Sublasses tui.Input and accepts any text but additionally provides tab completion for files and directories.
  • FloatInput.md - Sublasses tui.Input and only permits digits and decimal points to be entered by the user.
  • Input.md - Base class for various Text input types, which accept user input from the console.
  • NumberInput.md - Sublasses `tui.Input` and only permits digits to be entered by the user.
  • Popup.md - Popup text, a text bubble with some additional info for the user.
  • Progress.md - A progress bar, to indicate progress in an on-going operation.
  • SpinnerInput.md - Maeks a list of options selectable with upa nd down keys without the use of a cursor, display is only on one line, as the navigation keys are pressed the list is scrolled through.
  • Table.md - Renders a table of data.
  • Throbber.md - A Spinning throbber, ala Firefox, to indicate an on-going operation.
  • TimeInput.md - Sublasses tui.Input and only permits time to be entered in the form `00:00`.
  • ValidatedTextInput.md - Sublasses `tui.Input` to accept text entered by the user but validated by a function provided by a script.
