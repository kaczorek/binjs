Throbber

A Sppinning throbber bar, to indicate an ongoing operation.

The Object should be constructed and used only once, create a new Throbber with the new keyword to start the throbber again.

Import

binjs_import("~lib/tui/Throbber.js")

Provides

  • tui.Throbber

Constructor

The constructor takes two arguments.

var progress = new tui.Throbber(text, unicode);

text

The text to be displayed in fromt of the spinning icon.

unicode

When true unicode characters are used to render the throbber., when fals ascii caracters are used.

Attributes

text

The text to be drawn before the spinnign icon.

asciiGlyphs

The character used for the throbber in ascii mode ['\\', '|', '/', '-'].

the glyphs can be changed but theremust be 4 provided.

unicodeGlyphs

The characters used for the throbber in unicode mode ['·', '✶', '❆', '✺']

Methods

render

Draw the throbber untill a function returns true indicating the throbber can finished

throbber.render( function() {

    return new File("flag.file").exists;

} );.
