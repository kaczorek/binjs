API Index

Documentation for libraries that can be loaded with binjs_import() from ~lib/
  • Assert.md - Assert is a CommonJS like library for unit testing.
  • Color.md - Definitions for constants for coloured output on the terminal.
  • Dollar.md - The `$` Object is instantiated as an object in the global scope to provide some core utilities such as `$.println()` and `$.getEnv()`.
  • Dom.md - XML Dom parser and Document model.
  • File.md - File operations, for simple synchronous I/O.
  • Global.md - Global variables and functions.
  • Json.md - Syntactic sugar around v8's native JSON parser and a pretty printer.
  • Runtime.md - Runtime allows the script to interface with the environment in which it is running.
  • Sax.md - XML Sax parser.
  • Term.md - Provides access to features of the terminal the script is running in, such as terminal width, height and cursor position.
